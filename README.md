## NAME 
UHPy

## Overview
PythonからUnlimitedhandのデータを取得するためのツール


## Description
Unlimitedhandの角度とクォーニオン、フォトリフレクタの値の取得と電気刺激を流すことができる。

## Requirement
Unlimitedhandに対してSerial_Unity_QuaternionをArduinoに書き込んだ状態で使用してください。

Python3での動作
事前にpyserialのインストールが必要
pip install pyserial
